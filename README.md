## ✨ Featured Segments
* **Scale & Vision:** Monumental developments and urban planning.
* **Infrastructure Experts:** Highlighting civil engineering and public works.
* **Heavy Machinery:** Showcasing site preparation and excavation equipment.
* **Precision Metalwork:** Industrial welding and structural steel fabrication.

## 🛠️ Technical Stack
* **HTML5:** Semantic structure for industrial portfolios.
* **CSS3:** Custom animations, overlays, and responsive layouts.
* **Vanilla JavaScript:** Core logic for the carousel navigation and auto-transition.
